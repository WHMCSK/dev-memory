# 安装jupyter notebook

### 安装notebook
```
conda install notebook
```

### 安装扩展包
```
conda install -c conda-forge jupyter_contrib_nbextensions jupyterlab
```

### 启动jupyter

```
jupyter notebook
```

### 常规操作

配置jupyter notebook密码：

```
jupyter notebook password
```

启动jupyterlab：

```
jupyter lab
```

打开浏览器，输入`http://localhost:8888`访问jupyterlab。
