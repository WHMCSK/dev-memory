## 检查版本

如果已安装可以检查版本
```
protoc --version  // 检查Protocol Buffer Compiler的版本
protoc-gen-go --version // 检查Go support for Protocol Buffers版本
protoc-gen-go-grpc --version // 检查go grpc代码生成插件版本
```
